# Trainer user manual

## Admin workflow

1. Log in with an admin Supabase Auth user linked to a `profiles` row with role `admin`.
2. Open `/admin`.
3. Create or invite customers.
4. Add exercises to the library.
5. Create workout plans and assign them to customers.
6. Review booking requests and accept, decline, or propose alternatives.
7. Mark payments completed to unlock credits.
8. Complete sessions to consume credits.
9. Monitor audit logs and notifications.

## Customer workflow

1. Accept the Supabase invite email or log in after an admin creates the account.
2. Open the dashboard.
3. Review the next session, active workout plan, credit balance, payments, and notifications.
4. Request a slot from Availability.
5. View session detail and exercise instructions.
6. Update language and theme in Settings.
